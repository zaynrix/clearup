# Assets Organization

This directory contains all static assets used in the application.

## Structure

```
public/images/
├── backgrounds/     # Background images
│   └── background.png
└── logos/          # Logo files
    └── logo-main.png
```

## Usage

- **Background images**: Use `/images/backgrounds/[filename]` in your components
- **Logos**: Use `/images/logos/[filename]` in your components

All paths are relative to the `public` directory, so they can be accessed directly via URL.
